# Agent test projekat

Ovaj folder služi **isključivo za testiranje VajbAgent ekstenzije**.

- Svi zahtevi u chatu koji se odnose na "agent-test", "test projekat" ili izmene u ovom folderu — rade se samo ovde.
- Glavni projekat (vajbagent-vscode, src, backend, docs) — **ne dirati**.
- Možeš slobodno brisati, menjati ili dodavati fajlove u ovom folderu; to ne utiče na ostatak repozitorijuma.

## Kako testirati

1. Otvori VajbAgent chat.
2. Npr.: "U folderu agent-test napravi mi malu demo stranicu sa dugmetom" ili "Dodaj u agent-test/ novu stranicu contact.html".
3. Agent će raditi samo u `agent-test/` ako to eksplicitno pomenes.
