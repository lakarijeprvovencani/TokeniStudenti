VajbAgent je besplatna ekstenzija za Cursor i VS Code. Čita, piše i menja fajlove, pokreće komande, pretražuje web, pravi planove. Platiš samo koliko koristiš.

Mogućnosti
Sve što VajbAgent može da uradi
Kompletna lista mogućnosti ekstenzije — prostim jezikom.

Chat sa AI agentom

Pričaš sa agentom u panelu editora. On čita tvoje fajlove, piše novi kod, menja postojeći, pokreće komande — sve kroz konverzaciju. Streaming odgovori u realnom vremenu.

Plan Mode

Za složenije zadatke, agent napravi plan sa fazama. Ti kontrolišeš tempo — izvršavaš fazu po fazu ili sve odjednom. Pregled pre izvršavanja.

Checkpoint & Undo

Svaka izmena se čuva. Ako ti se ne svidi rezultat — jedan klik i vratiš sve na prethodno stanje. Po fajlu ili sve odjednom.

Čita i piše fajlove

Automatski čita, kreira i menja fajlove u tvom projektu. Diff preview pre svake izmene.

Pokreće komande

Pokreće shell komande u terminalu — npm install, git commit, build, test... Sve iz chata.

Slash komande

/test · /fix · /doc · /commit · /explain · /refactor — brze akcije jednim klikom.

@file i @folder mentions

Taguješ @fajl ili @folder u chatu — agent odmah čita sadržaj. Autocomplete ti nudi fajlove.

Web pretraga

Agent pretražuje internet za aktuelne informacije — dokumentaciju, rešenja, primere.

Fetch URL

Daj mu link — pročitaće sadržaj stranice, API odgovor, dokumentaciju. GET, POST, PUT, DELETE.

Slike & drag-drop

Prevuci sliku u chat, nalepi sa Ctrl+V, ili dodaj preko file picker-a. Agent vidi i analizira slike.

Pretraga slika (Unsplash)

Agent traži i skida besplatne stock fotografije sa Unsplash-a direktno u tvoj projekat.

MCP integracija

Povezivanje sa eksternim servisima — Supabase, GitHub, baze podataka, Slack i bilo koji MCP server.

Auto-approve mode

Uključi auto-approve za write, replace ili execute — agent radi bez pitanja. Ili odobravaj svaku akciju pojedinačno.

CONTEXT.md

Agent pamti tvoj projekat između sesija. Struktura, odluke, stack — sve sačuvano u jednom fajlu.

Dijagnostika

Vidi greške iz VS Code-a u realnom vremenu. Posle svake izmene proverava da li je nešto puklo.

PDF čitanje

Prikači PDF u chat — agent čita sadržaj i koristi ga za kontekst. Dokumentacija, specifikacije, zadaci.

Git operacije

Commit, push, branch, diff — sve iz chata. /commit komanda generiše poruku iz promena.

@terminal mention

Taguješ @terminal — agent čita poslednji output iz terminala. Savršeno za debugging build grešaka.

Custom pravila

Napravi .vajbagentrules fajl — agent prati tvoja pravila (TypeScript strict, pnpm, srpski komentari...).

Istorija sesija

Svi prethodni razgovori sačuvani. Vrati se na bilo koji, nastavi gde si stao ili obriši.

Explain / Refactor

Selektuj kod, desni klik → Explain ili Refactor. Ili koristi slash komande /explain i /refactor.

Markdown & syntax highlight

Odgovori sa formatiranim kodom, tabelama, listama. Copy dugme na svakom code bloku.

Jedan ključ, svi modeli

Jedan VajbAgent API ključ — pristupaš svim modelima. Bez naloga kod više provajdera.

Lite — svakodnevno
