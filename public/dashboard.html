🌐 Web pretraga za agenta (besplatno)

Dodaj web pretragu svom agentu u Cline-u. Potpuno besplatno — bez API ključa, bez naloga. Agent će moći da pretražuje internet kad mu zatreba.

1
U VS Code-u mora da ti bude instaliran Node.js (proveri sa node -v u terminalu). Ako nemaš, preuzmi sa nodejs.org
2
Otvori Cline panel → ⚙️ Settings → skroluj do MCP Servers → klikni "Configure" (ili "Edit MCP Settings")
3
Otvoriće se fajl cline_mcp_settings.json. Zameni ceo sadržaj ovim:
{
  "mcpServers": {
    "web-search": {
      "command": "npx",
      "args": ["open-websearch@latest"],
      "env": {
        "MODE": "stdio",
        "DEFAULT_SEARCH_ENGINE": "duckduckgo"
      }
    }
  }
}
4
Sačuvaj fajl (Ctrl+S). Cline će automatski pokrenuti MCP server.
5
Testiranje: u chatu pitaj agenta nešto što zahteva internet, npr. "Pretraži šta je novo u React 19". Cline će ponuditi web_search alat — odobri ga i agent pretražuje!
Napomena: Ako već imaš nešto u cline_mcp_settings.json, samo dodaj "web-search" objekat unutar postojećeg mcpServers. Na Windows-u koristi "command": "cmd" i "args": ["/c", "npx", "open-websearch@latest"].
