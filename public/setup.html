Podešavanje u Cursoru

Sve informacije za povezivanje VajbAgent-a sa tvojim Cursorom.

Override Base URL

Ovo je URL koji unosiš u Cursor podešavanja. Sačuvaj ga ili se vrati na ovu stranicu kad god ti treba.

Korak po korak

1
Otvori Cursor → ikona zupčanika (Settings) dole levo → Cursor Settings.
2
Idi na Models → proširi sekciju OpenAI API Key.
3
U polje API Key unesi tvoj VajbAgent ključ (ne Anthropic ključ, nego tvoj lični ključ koji si dobio).
4
Uključi Override OpenAI Base URL i unesi URL iznad.
5
Klikni "+ Add Model" i dodaj jedan od modela iz tabele ispod (npr. vajb-agent-pro).
6
U chatu izaberi dodani model i pošalji poruku. Ako dobiješ odgovor, sve radi!
Važno: Dok je Override uključen, Cursor će koristiti tvoj VajbAgent API za izabrani model. Kad hoćeš da koristiš Cursor-ove ugrađene modele, isključi Override toggle. URL ostaje sačuvan za kad ponovo uključiš.
