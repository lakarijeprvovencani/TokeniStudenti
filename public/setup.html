Korak po korak

1
Otvori Cursor → ikona zupčanika (Settings) → Cursor Settings
2
Idi na Models → proširi sekciju OpenAI API Key
3
U polje API Key unesi tvoj VajbAgent ključ (ne Anthropic ključ, nego tvoj lični ključ koji si dobio)
4
Uključi Override OpenAI Base URL i unesi URL iznad
5
Klikni "+ Add Model" i dodaj model iz tabele ispod (npr. vajb-agent-pro)
6
U chatu izaberi dodani model i pošalji poruku. Ako dobiješ odgovor, sve radi!
Važno: Dok je Override uključen, Cursor koristi tvoj VajbAgent API. Kad hoćeš ugrađene modele, isključi Override toggle. URL ostaje sačuvan.
